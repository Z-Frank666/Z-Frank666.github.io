python学习阶段总结

个人阶段性总结

接近四周（有效学习时间其实不足一半）的python自学在今天终于落下帷幕，对于个人来说，这几周的学习给我的直观感受是python这门语言的高度简约性和封装性，同时其完备的各项设计又使其在各大应用领域发挥举足轻重的作用，但惭愧的是，本人自学python终究只能做到了解的程度，希望以后能通过项目和学校课程的学习进一步完善自己。

python的一些创新设计

学习python的过程中，我个人对一些设计印象深刻，而这些设计对于初学者来说大概率会显得相对难以快速接受（对于C/C++入门者），因此我将对这些设计写一点个人的理解，也算是一次知识复盘。

1、python函数进阶

（1）函数多种传参方式

（2）函数多返回值

学了C/C++就知道，函数没有多返回值这一设计，而python的多返回值设计完美弥补了这一点。

3、正则表达式

感兴趣的朋友可以自行运行，但这里我还想着重讲解一下最后一个案例：

首先，正则表达式的精华在于元字符匹配，而元字符匹配的精华又在于格式化配置变量的构建

对于一个常规邮箱地址，可以分为三个部分：@以前，@与.(dot)之间，.(dot)之后。

首先^与$对应了首位匹配，实际上是匹配范围限制作用，而@之前的部分除了非特殊字符之外，可能会有额外.(dot),因此加入了(.[\w-]+)*意即(.xxx)部分存在0到无数个。而.com之后又可能还有.cn等出现，因此需要使用+而不是简单的一次。

总结及后续规划
